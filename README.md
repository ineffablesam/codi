# Codi

Codi is a large-language-model agent that can be used to manipulate code repositories. It integrates with GitHub issues and PMs. It is currently under development.

## Installation

We commend using a virtual environment for development. Not only does it improve security by insulating your project from the operating system but it also ensures that libraries do not conflict. You can refer to the Python documentation on how to set up a virtual environment.
