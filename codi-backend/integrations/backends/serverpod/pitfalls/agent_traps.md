---
- title: "Don't manually implement authentication"
  description: "Agents often try to build custom JWT/session logic. Serverpod has built-in auth."
  wrong: "Creating custom JWT tokens and manual session management"
  correct: "Use `serverpod_auth` package and extend `AuthenticationInfo`"
- title: "Don't create REST endpoints manually"
  description: "Serverpod uses RPC, not REST. Every endpoint method is automatically exposed."
  wrong: "Defining routes like `/api/users/:id`"
  correct: "Create method in Endpoint class: `Future<User?> getUser(Session session, int id)`"
- title: "Don't write SQL queries manually"
  description: "The ORM is type-safe and prevents SQL injection"
  wrong: "session.db.query('SELECT * FROM users WHERE id = $userId')"
  correct: "User.db.findById(session, userId)"
- title: "Don't forget to regenerate after protocol changes"
  description: "LLMs often forget this crucial step, causing compile errors"
  wrong: "Editing protocol/*.yaml and immediately trying to use the model"
  correct: "Edit protocol/*.yaml → run `serverpod generate` → use model"
---
